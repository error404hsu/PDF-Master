# PDF 排列哥 — 後續實作清單（辦公室合併）

以下項目在介面或 `ExportOptions` 中可能已有欄位或需求描述，但**尚未完整實作**或僅部分支援。實作時請同步更新本檔與使用者說明。

## 匯出與文件結構

- **保留書籤（TOC）**：`ExportOptions.keep_bookmarks` — 合併時依規則重對應或合併多份 PDF 的書籤樹（PyMuPDF `insert_pdf` / `set_toc` 等）。
- **保留附件**：`keep_attachments` — 將來源 `embfile` 複製到輸出文件。
- **保留互動表單**：`keep_forms` — 合併含 AcroForm 的頁面時避免表單欄位名稱衝突，或提供「扁平化」選項。
- **子資料夾遞迴**：「開啟資料夾」目前僅掃描一層；可選「包含子資料夾」與檔名排序規則。
- **匯出前完整檢查報告**：除加密提示外，列出缺字型、非內嵌字型、頁面尺寸不一致等（需擴充 `inspect_pdf`）。

## 編輯與工作階段

- **工作階段存檔／還原**：將頁序、旋轉、來源路徑（建議相對路徑）存成專案檔，方便辦公室重開接續處理。
- **複製／貼上頁**：同一或不同來源之重複頁面（新 `page_id`）。
- **插入空白頁／章節分隔頁**：向量空白頁或範本頁。
- **清單檢視模式**：大量頁面時切換為緊湊列表 + 小縮圖。

## 進階（選做）

- **浮水印／頁碼 stamp**：合併後批次加上機密等級或「第 x / y 頁」。
- **裁切框（Crop）視覺編輯**：於縮圖上調整可見區再匯出。
- **監看資料夾／掃描佇列**：新 PDF 自動進入工作區。
